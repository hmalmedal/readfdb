Read FDB Files
==============

Read files from [Ferjedatabanken](http://fdb.triona.no/).

Installation
------------

Install from GitHub:

``` r
devtools::install_github("hmalmedal/readfdb")
```

Alternative method:

``` r
install.packages("https://github.com/hmalmedal/readfdb/archive/master.tar.gz",
                 type = "source", repos = NULL)
```
